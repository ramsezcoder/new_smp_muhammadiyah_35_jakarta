### Admin Access
```
URL: http://localhost:3001/admin

Credentials:
- Superadmin: SuperAdmin@2025 (full access)
- Admin: Admin@2025 (read-only)
- Post Maker: PostMaker@2025 (no gallery access)
```

### Development
```bash
npm install && npm run dev              # Frontend
cd server && npm install && npm run dev # Backend (separate terminal)
```

### Production
```bash
npm run build
cd server && npm start
```

### Key Routes
```
/ - Home
/news - News list
/news/:slug - News detail
/gallery - Gallery index (NEW)
/gallery/photos - Photo gallery
/gallery/videos - Video gallery
/gallery/infographics - Infographics
/student/e-module - PDF module
/admin - Admin panel (NEW CMS)
```

### API Endpoints
```
POST /api/upload/gallery - Upload image (superadmin only)
GET /api/news/list - Get news
GET /api/pdf/views - Get PDF views
POST /api/pdf/view/:id - Increment PDF view
```
